# Simulador Dinamômetro Moto

Simulador web interativo para dinamômetro com freio hidráulico, rolo, sensor de rotação e célula de carga.

## Como rodar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy no Railway
1. Vá para [Railway.app](https://railway.app)
2. New Project → Deploy from GitHub Repo
3. Selecione este repositório
4. Deploy automático!

Acesse a URL gerada para usar o simulador online.
